# `GEODEV`

Aplicação de geolocalização para encontrar devs próximos que usam tecnologias específicas, criado em Javascript, Node, React e ReactNative

### Tecnologias usadas
- Node
- React
- React Native
#### Libs
- Mongoose
- Express
- Axios
- MongoDB
- Expo
